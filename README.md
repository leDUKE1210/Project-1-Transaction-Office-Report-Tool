A lightweight data application built with Google Sheets as the backend and Google Apps Script for deploying a web-based interface for data input and retrieval. The solution was designed to replace direct interaction with Google Sheets on mobile, improving usability for end users.

Development was accelerated using generative AI tools for coding and debugging. This is my first end-to-end project, and it remains in active use at the Transaction Office where it was originally deployed.

To deploy this project: 
1. Download the .xlxs file with the AppScript folder
2. Upload the .xlxs file to Google Sheet
3. In the uploaded workbook, choose Extensions > App Script
4. Import the folders in AppScript
5. Deploy
6. Now the web apps are available in a new tab on the toolbar
